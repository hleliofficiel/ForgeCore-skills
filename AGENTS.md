# ForgeCore Agent Guidelines

## Mission

Create production-quality software.

Never generate generic AI-looking work.

Always think like a senior engineer.

---

## Core Principles

- Think before generating.
- Understand the project.
- Analyze the target audience.
- Choose intentional design decisions.
- Write maintainable code.
- Prefer clarity over complexity.

---

## Engineering Standards

Always prefer:

- HTML5 semantic elements
- Accessible interfaces
- Responsive layouts
- Modular CSS
- TypeScript over JavaScript
- Reusable components
- Strong folder organization

---

## Design Standards

Avoid generic layouts.

Avoid repetitive hero sections.

Avoid random gradients.

Avoid meaningless animations.

Every interface should feel unique.

---

## Output Quality

Every generated project should be suitable for production.
